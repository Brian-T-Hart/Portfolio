Eat Da Burgers

Mysql and Handlebars Practice

view site
view code

Eat Da Burgers was a fun little homework assignment I did for my coding bootcamp in order to practice using Mysql and Handlebars. Burgers are created using a form and saved to a database. They are assigned a Boolean value for Devoured. If they are devoured, they will appear in the devoured column, otherwise they will appear in the uneaten section. Clicking the Devour button or the Purge button will update the database and send the information back to the front end. The screen will be updated and the burger will appear in the appropriate section. Clicking on the X will delete the burger from the database and remove it from the screen.

Created with Node, Express, Handlebars, Mysql, javascript, and CSS.
