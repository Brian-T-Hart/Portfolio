Goal-Post-CRM

ON-THE-GO SALES TOOL

view site
view code

Goal-Post-CRM was the final project I worked on at The Coding Bootcamp at UCI. I was one of a team of five members that worked on creating a full-stack project utilizing much of what we learned throughout the 24 week intensive training. The project's front-end was built using React. The back-end utilized Node to bring in a number of packages, such as, Express, Passport, and Sequelize. We used a Mysql database.

I worked on a number of different aspects of the project. I helped to develop the models for the database. I created many of the database queries. I set up Passport and utilized a Json Web token for authentification. I helped work out the API calls on the front-end. I also helped with the set up of the chart on the dashboard.
